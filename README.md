# Structure-Vanilla-Php

## Clone code
```
git clone https://github.com/Thenam2kx/Structure-Vanilla-Php.git

```


## Cài đặt frontend

```
1. cd Frontend
2. yarn
3. yarn dev
4. truy cập (http://localhost:3000)
```

## Cài đặt backend

```
1. cd Backend
2. composer install
3. php -S localhost:8000 -t backend
4. truy cập (http://localhost:8000)

```
